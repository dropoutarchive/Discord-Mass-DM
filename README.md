### Features

* Fast
* Asynchronous
* Easy to use
* Free
* Auto Scrapes User

* Server Joiner and Leaver
* Multiple Tokens
* Blacklist Users
* Avoid DMing Bots
* Keep Tokens Online [OPTIONAL]
